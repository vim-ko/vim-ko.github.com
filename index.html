빔 편집기 한글화

빔(Vim) 편집기의 도움말 및 이와 관련된 여러 자료를 우리말로 번역하고 있습니다.
  • 온라인 도움말에서 지금까지 번역된 도움말을 웹브라우저로 바로 보실 수 있습니다.
  • 빔 편집기에서 한글 도움말을 사용하시려면, 우선 한글 도움말 묶음을 받으신 다음, doc 디렉토리가 ~/.vim/ 아래에 있도록 압축을 풀어두시면 :help 명령이 한글 도움말을 보여줍니다. 여전히 도움말이 영문으로 나온다면, :set helplang=ko 명령으로 언어를 한국어로 바꿔주십시오.

vim-ko 모임

번역은 여러 사람들의 자발적인 참여로 진행하고 있습니다. 처음에 구글 코드에 둥지를 틀고 진행을 조금했지만, 최근에는 점차 GitHub로 이사를 하고 있습니다. 함께 번역하는 일에 동참하실 분은 우선 구글 그룹스에 가입하시고, GitHub 저장소를 fork하시고 시작하시면 됩니다.
  • vim-ko 구글 그룹스
  • vim-ko GitHub 저장소 (구글 코드 프로젝트)
